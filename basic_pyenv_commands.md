# pyenv

Посмотреть установленные версии Python.

```commandline
pyenv versions
```

Посмотреть какая версия используется в текущей дирректории.

```commandline
pyenv version
```

Установить какая версия Python будет в текущей директории.

```commandline
pyenv local <version_python>
```

Установить необходимую версию Python.

```commandline
pyenv install <version_python>
```
